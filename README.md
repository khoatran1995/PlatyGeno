# PlatyGeno
PlatyGeno is an interpretability-driven discovery tool for the Evo 2 foundation model. It utilizes Sparse Autoencoders (SAE) to map high-dimensional latent activations to verifiable biological motifs, enabling the discovery of functional elements in unannotated genomic data.










Acknowledgements

This tool is built upon the Evo 2 genomic foundation model developed by the Arc Institute. We are grateful to the Arc Institute and the Goodfire team for providing the model weights and sparse autoencoder (SAE) architectures that make PlatyGeno possible.
