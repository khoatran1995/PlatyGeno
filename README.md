# PlatyGeno
PlatyGeno is an interpretability tool for the Evo 2 genomic foundation model.


🛠 Dependencies
To run PlatyGeno, users must have the following installed (these are not included in this repository, but will be guided below):
Evo 2 Foundation Model (Arc Institute)
Sparse Autoencoder Weights (Goodfire)










Acknowledgements

This tool is built upon the Evo 2 genomic foundation model developed by the Arc Institute. We are grateful to the Arc Institute and the Goodfire team for providing the model weights and sparse autoencoder (SAE) architectures that make PlatyGeno possible.
